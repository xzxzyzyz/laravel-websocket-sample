## Laravelでwebsocket通信のサンプル

* Laravel
* Redis
* Node.js
* Vue.js

1. `$ git clone https://github.com/xzxzyzyz/laravel-websocket-sample.git && cd laravel-websocket-sample.git`
2. `$ composer install`
3. `$ npm install`
4. setting `.env`
5. `$ php artisan serve`
6. `$ php artisan queue:listen`
7. `$ node server.js`
